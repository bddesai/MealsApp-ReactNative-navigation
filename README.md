Prerequisites 
1. Yarn 
2. NPM
3. React Native 
4. Expo 


Steps to install 
1. Run `yarn install`
2. Run `yarn start` to start the server 
   1. For android - `a`
   2. For iOS - `i`

3. If you want to run on the platform specific emulator directly, then run `yarn android` or `yarn ios`
